# مرحلة التحليل  
## نظام إنشاء الجداول الأكاديمية  
 

# 1. المتطلبات الوظيفية (Functional Requirements)

## 1.1 إدارة البيانات الأساسية  
يجب أن يسمح النظام بالعمليات التالية:  
- إضافة / تعديل / حذف **المقرر**  
- إضافة / تعديل / حذف **الدكتور النظري**  
- إضافة / تعديل / حذف **المهندس العملي**  
- إضافة / تعديل / حذف **القاعة**  
- إضافة / تعديل / حذف **القسم**  
- إضافة / تعديل / حذف **المستوى الأكاديمي**  
 
---

## 1.2 توليد الجدول  
- يقوم النظام بتوليد جدول أسبوعي مبسط بناءً على:  
  - القسم  
  - المستوى  
- التوزيع بسيط دون خوارزميات معقدة.  
 
---

## 1.3 عرض الجدول  
يعرض النظام الجدول بالحقول:  
- اليوم  
- الوقت  
- المقرر  
- القاعة  
- الدكتور النظري  
- المهندس العملي  
- القسم  
- المستوى  
 
---

## 1.4 تصدير الجدول  
- يمكن تصدير الجدول إلى PDF  
- مع دعم RTL  
- بتنسيق بسيط جاهز للطباعة  
 
---

## 1.5 حفظ البيانات  
- حفظ الجدول الناتج في قاعدة بيانات SQL Server  
 
---
